# Udacity project to Check the weather in certian countries

## Built With:

- Front-end:
  - HTML
  - Sass
  - JS
  - Webpack
  - Jest
- server :
  - nodeJS

## How to use it :

- Install the packages using npm i
- run 2 terminals :

  - npm run build-dev
  - npm run vuild-prod npm run start

- Write the place you are going to visit
- insert the dates
- Click on Check button
